What is a profile?

A profile is a medication record for a single individual, containing a list of drugs, conditions, and allergies applicable to that person.

Multiple profiles can be setup on your account. For example, you could setup a profile for yourself and another one for your spouse, your mother, a friend, etc.

How to add a new profile
  • tap the Profiles link (in the left-hand column) to view the list of profiles.
  • tap on the name of the profile you want to switch to.
How to delete a profile
  • tap the Profiles link (in the left-hand column) to view the list of profiles.
  • tap on the name of the profile you want to switch to.
  • Tap on Add Profile

Enter a name to identify the profile ("John", "Mom", etc), add further details, and press the Save button when complete.

How to change to a different profile
  • tap the Profiles link (in the left-hand column) to view the list of profiles.
  • Tap on the name of the profile you want to switch to.
How to delete a profile
  • tap the Profiles link (in the left-hand column) to view the list of profiles.
  • Tap on the name of the profile you want to switch to.
  • Tap on Edit Profile
  • On the profile edit page, scroll down and tap on the red "Delete" button.
How to add a new drug to your profile
  • Tap on Drugs(in the left-hand column)to view your list of medications.
  • Tap on Add Drug

Type the first few letters of the drug name and a list of suggestions should appear. Scroll down to see more options or continue typing the name of the drug to narrow the list. Once found, tap on the matching drug.
You will be redirected to the drug edit page where further details can be specified, such as the condition being treated, dosage details, or whether you'd like to receive email notifications relevant to this drug.
Press the Save button when finished.

How to remove (delete) a drug from your profile
  • Tap on Drugs(in the left-hand column)to view your list of medications.
  • Select (tap on) the drug you want to remove.
  • On the drug details page, tap on the "Edit drug" link.
  • On the drug edit page, scroll down and tap on the red "Delete" button.
How to add a new condition to your profile
  • Tap on Drugs (in the left-hand column) to view your list of conditions
  • Tap on Add Condition

Type the first few letters of the condition name and a list of suggestions should appear. Scroll down to see more options or continue typing the name of the condition to narrow the list. Once found, tap on the matching condition.
You will be redirected to the condition edit page where further details can be specified, such as whether you'd like to receive email notifications relevant to this condition.
Press the Save button when finished.

How to remove (delete) a condition from your profile
  • Tap on Drugs (in the left-hand column) to view your list of conditions
  • Select (tap on) the condition you want to remove.
  • On the condition details page, tap on the "Edit condition" link.
  • On the condition edit page, scroll down and tap on the red "Delete" button.
How to setup profile sharing

The profile sharing page allows you to view and change your profile's sharing status (private/public). This page also contains more detailed information on how profile sharing works.

How to email a profile
  • Tap the profile sharing
  • Ensure the "Sharing Status" option is set to "Sharing Status"
  • Tap on "Email share link" to launch your default email program.

The profile sharing also contains more detailed information on how profile sharing works and what other options are available.

How to print a profile
  • Tap the print option (in the left-hand column) to view the profile printing page
  • Check (or uncheck) any desired Print Options (i.e. choose to include/exclude notes, researching drugs, previously taken drugs, etc).
  • Select one of the three layout choices: Summary, Complete, Wallet-sized.
  • On the print layout page, scroll down and tap on the blue "Print" button.
Drug Interaction Classifications

These classifications are only a guideline. The relevance of a particular drug interaction to a specific individual is difficult to determine. Always consult your healthcare provider.

  • Major: Highly clinically significant. Avoid combinations; the risk of the interaction outweighs the benefit.
  • Moderate : Moderately clinically significant. Usually avoid combinations; use it only under special circumstances.
  • Minor : Minimally clinically significant. Minimize risk; assess risk and consider an alternative drug, take steps to circumvent the interaction risk and/or institute a monitoring plan.
  • Minor: No interaction data available.
